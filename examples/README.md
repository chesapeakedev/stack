# Examples

Copy-pasteable React examples demonstrating `@chesapeake/stack` components and
patterns. Each file is self-contained — drop it into any React app that has the
stack installed and configured per the [root README](../README.md).

## Examples

| File | Description |
| --- | --- |
| `basic-form.tsx` | Form built with Input, Label, Textarea, Select, and Button |
| `auth-setup.tsx` | Multi-provider authentication with Google and GitHub |
| `page-layout.tsx` | App shell using StandardLayout, PageLayout, and PageHeader |
| `theme-customization.tsx` | ThemeProvider with custom theme overrides |
| `feedback-and-notifications.tsx` | FeedbackDrawer, toast notifications, and Dialog |
| `data-display.tsx` | Cards, badges, avatars, tooltips, and dropdown menus |
| `user-management.tsx` | UserSelector and FriendList for social features |
